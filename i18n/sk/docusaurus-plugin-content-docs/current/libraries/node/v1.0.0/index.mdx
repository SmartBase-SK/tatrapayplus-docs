# Getting started

To be able to use and test payment gateway you need to have **Client ID** and **Client secret** from [TB developer portal](https://developer.tatrabanka.sk/).
To obtain these tokens you need to register in developer portal, for more info, please visit [Developer portal documentation](https://developer.tatrabanka.sk/apihub/#/apis/f438e906-7845-4bc1-a028-73d3f9547d2a/show/doc?mode=view&uri=Ako-za_a_).

# Installation

To install API client you need to install `tatrapayplus-node` package.
```npm
npm i @tatrabanka/tatrapayplus-node
```
